# Round a number to defined digital places

If the number is less than 10, it returns it rounded with 3 digits, if
it's less than 100 with 2 digits, if it's less than 1000 with 1 digit,
otherwise with 0 digits.

## Usage

``` r
round_the_result(x)
```

## Arguments

- x:

  a single number.

## Value

a single number.
